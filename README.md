# Elewachi Emmanuel — Portfolio

Personal portfolio website for Elewachi Emmanuel, AI Automation Expert.

## Getting Started

```bash
npm install
npm run dev
```

The dev server runs at `http://localhost:8080/`.

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Shadcn/ui
